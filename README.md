# CR Soundboard

This the AHS Coding Club's first app, a Clash Royale soundboard made entirely with SwiftUI.

## App Store!

The soundboard is [now available on the Apple App Store](https://apps.apple.com/us/app/cr-soundboard/id6443834152)!
